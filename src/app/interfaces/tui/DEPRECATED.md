# TUI is deprecated

The Textual TUI (`mova tui`, `src/app/interfaces/tui/`) is **no longer maintained**.

- **Use the web UI** instead: `mova` (serves `http://127.0.0.1:8000` by default).
- **Do not add new features** to the TUI — implement them in `frontend/` and `backend/` only.
- This package **will be removed** in a future release.

CLI (`mova cli`) remains supported.
